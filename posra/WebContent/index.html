How does P-OSRA work?

First, you upload an image of a polymer. This image is submitted to our polymer image recognition software, and a SMILES string is returned.

Second, that SMILES string is processed and placed into a database which keeps track of all previously submitted polymers. We are working on end-user querying capabilities.

p-osra is an open-source project funded and run by developers at IBM.
