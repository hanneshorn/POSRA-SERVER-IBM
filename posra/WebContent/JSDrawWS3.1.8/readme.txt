Install
1. Create a Web Application under IIS/ASP.NET
2. Set Application Pool to .NET 2.0 Classic
3. Place license code in web.config ( Please contact us to get one: http://www.scilligence.com/web/contact.aspx )
4. Please download MolEngine Plugins to extra features, such as InChI
