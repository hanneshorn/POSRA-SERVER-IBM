MolEngine Plugins includes following modules:

1. Name-to-Structure OPSIN ( http://opsin.ch.cam.ac.uk/ ), wrapped with IKVM ( http://www.ikvm.net/ )
2. InChI ( http://www.iupac.org/home/publications/e-resources/inchi.html )
3. Smi23D ( http://www.chembiogrid.org/cheminfo/smi23d/ )


Installation:

1. Copy the whole Plugins folder to C:\Program Files\Scilligence\


Requirement

1. VC++ 2008 runtime has to be installed, which can be downloaded at: 
        http://www.microsoft.com/en-us/download/details.aspx?id=2092
